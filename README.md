Flash Cards application
This application presents the user with statements and the user is required to choose if the statement presented is either a myth or life hack based on their general knowledge and real life experiences.
The app has a welcome screen that has a "Start" button that allows the user to begin.
The flash cards have three buttons:
Myth button
Hack button
Next button
When the user selects the "Myth" button it means that they think the above statement is false, and if they click on the "Hack" button it means that they the above statement is indeed a life hack.
After selecting between "Myth" and "Hack" the user is required to click "Next" to lock in their answer (the user cannot continue without choosing an answer on the current flashcard).
When the user has gone through the whole flashcard stack their answers are graded and the user is taken to the score card when they can see their score out of 10 and aslo see where they went wrong as the wrong answers are highlighted in red and the correct ones in green.
The score card page also has a "Restart" button at the bottom of the page for if the user wishes to start again.
